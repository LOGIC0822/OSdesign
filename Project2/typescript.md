# Build
Use
```shell
make
```
in each directory (named BurgerBuddies and Santaclaus),
executables `BBC` and `Santaclaus` will be made out in the corresponding directories. You can `cd BurgerBuddies` for example and test each program.

```shell
./BBC <cooks> <cashiers> <customers> <rack_num>
```
